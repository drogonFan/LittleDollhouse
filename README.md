## ChatRoom

An online chat room with cs architecture.Users need to register on the server. Users can communicate using p2p.

The specific supported functions are as follows:

+ User Registration / Logout
+ User login
+ Heartbeat detection
+ History chat
+ Private chat
+ group chat
+ p2p chat
+ p2p file transfer
